Test Header Behavior

Este es un test para verificar que el header se ajuste correctamente al sidebar.

Instrucciones:

  1. Haz click en "Toggle Sidebar" para contraer/expandir el sidebar
  2. Observa cómo el header se ajusta automáticamente
  3. Verifica que la transición sea suave
  4. Verifica que no haya desbordamiento

Comportamiento Esperado:

  • ✅ Header se expande cuando sidebar se contrae
  • ✅ Header se contrae cuando sidebar se expande
  • ✅ Transición suave entre estados
  • ✅ No hay desbordamiento de contenido
